# Phase-5: Full VAPT Cycle

## Objective
To perform a complete Vulnerability Assessment and Penetration Testing
(VAPT) cycle on a vulnerable web application in a lab environment.

## Tools Used
- Nmap
- OpenVAS
- sqlmap
- sha256sum

## Activities Performed
1. Vulnerability scanning using Nmap and OpenVAS
2. Vulnerability analysis and prioritization
3. Exploitation of SQL Injection using sqlmap
4. Evidence collection and hashing
5. Remediation recommendation and verification

## Deliverables
- Scan reports
- Exploitation proof
- Screenshots
- Remediation notes

## Status
Completed successfully
